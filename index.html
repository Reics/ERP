Proyecto para Sistemas de Información.

Este proyecto se desarrolla sobre codeigniter, bootstrap y MySQL.

Componentes necesarios para instalación.

  • Git
  • Apache
  • MySQL(instalación opcional)

Como clonar el proyecto.

Abrimos Git bash en windows o Terminal en linux

Entramos en la carpeta donde carga las paginas nuestro Apache ya sea en htdocs, www, html, etc.

Despues se utiliza el siguiente comando dependiendo su equipo:

Para el equipo de Pao:

git clone -b Precios_Unitarios https://github.com/Reics/ERP.git

Para el de Rafa:

git clone -b sistInfDespachoArq```uitectos https://github.com/Reics/ERP.git

Para el de Diana:

git clone -b proveedores https://github.com/Reics/ERP.git

Conexión a base de datos

Cuando tengan clonado el proyecto, dentro de la carpeta application\config agregan el archivo database.php y modifican ese archivo con los datos de su base de datos
